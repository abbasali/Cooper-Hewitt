Provides the Fancybox jQuery plugin for displaying images (HTML content and
multimedia support will come) in an elegant box.

Requirements
------------

1. jQuery 1.3.x - install jQuery Update 6.x-2.x <http://drupal.org/project/jquery_update>
2. Fancybox 1.3+ - download latest version from <http://code.google.com/p/fancybox/downloads/list> (1.3.1 is recommended)

Installation
------------

0. Install jquery_update 2.x

1. Place this module directory in your modules folder (usually sites/all/modules/).

2. Extract the plugin archive and copy folder with plugin files (containing jquery.fancybox-[version].js file) 
   in the following (recommended) location: sites/all/libraries

   So the actual plugin can be found at:
   sites/all/libraries/fancybox/jquery.fancybox-[version].js

3. Go to "Administer" > "Modules" and enable the module.

Configuration
-------------

"Site configuration" > "Fancybox"
